## IMPLEMENTATION NOTES

### Dart Data Structure
```dart
// lib/core/constants/india_data.dart
const Map<String, List<String>> indiaStatesDistricts = {
  'Andhra Pradesh': ['Alluri Sitharama Raju', 'Anakapalli', ...],
  'Arunachal Pradesh': ['Anjaw', 'Changlang', ...],
  // ... all 36 entries
};

// Sorted list for state dropdown
final List<String> indianStates = indiaStatesDistricts.keys.toList()..sort();

// Get districts for selected state
List<String> getDistricts(String state) =>
    indiaStatesDistricts[state] ?? [];
```

### UI Behavior
- **State dropdown:** Searchable, scrollable, alphabetical
- **District dropdown:** Appears ONLY after state is selected
- **Auto-detect:** Pre-select state based on SIM card region (if possible)
- **Search:** Fuzzy search for both state and district names
- **Vernacular:** Show state names in user's selected language where possible
